머신러닝

인공지능이란 말을 만들어냈을 때 1950년대
1980년대 퍼스널 컴퓨터가 사용되면서 머신러닝이란 말이 생겨남
데이터를 컴퓨터에 집어넣는거, 분석,해석을 수학으로 했음(알고리듬) 그 결과물을 모델 이라고 불렀음
여기서부터 머신러닝이라고 부르기 시작함 기계가 학습해서

머신러닝의 여러 장르 중 딥러닝이 시작됨
인공신경(AN)을 이용해 인공신경망(ANN)을 만들어 데이터를 흘려보냄
풀려는 문제마다 어떻게 인공신경망을 재구성하느냐가 중요함 

머신러닝 : 기계가 학습한다. 컴퓨터언어(=데이터)로 모델링해서 모델을 만든다. 
알고리듬으로 학습한다. 이걸 만드는걸 인간이 한다=데이터엔지니어
컴퓨터 과학 = a하고 b를 넣으면 a+b가 나오는 거
사람이 찾아서 기계한테 알려줌

머신 러닝 = a하고 b를 넣으면 c가 나오는데 c가 왜 나오는지 과정, 데이터의 원래 목적(모델)을 뽑아내는거
데이터 많이 주고 기계한테 직접 하게함

알고리즘 3가지
지도학습 ( Supervised learning ) : 문제(Feature)와 정답(Label)을 제공(Labeling)해주면 예측, 추정, 분류하는 알고리즘
(분류 : Classfication, 회귀 : Regression)
Unsupervised learning : 비지도학습, 문제(Feature)만 제공, 패턴이나 구조를 발견하여 그룹화
변칙처리(Anomaly)
Reinforcement learning : 강화학습
